# Unihack2025
Web Platform for scheduling and discovering of social gathering events inside the city of Timisoara created along side a team of 3 during UniHack2025


Used technologies:

  -frontend: JavaScript, Node.js;
  
  -backend: Python, Flask;
  
  -MongoDB

Functionalities:

  -Schedule and post social gathering events such as parties and concerts within the city of Timisoara
  
  -Discover and search for existing events
  
  -Google Gemini API implementation for a chat helper that can help you find the kinf of event you are looking for base on description
  
  -Google Gemini API implementation that checks for profanity and other obscene language within posts
  
  -Google Maps component for a better understanding of the location of existing events
